Integral, Matematika pelajaran BAB pertama pada kelas 12, Integral adalah lawan(invers) dari turunan. Jadi apabila kita menurunan suatu fungsi kita 'menaikannya' kembali dengan integral, contoh:

apabila kita mempunyai fungsi:
f(x) = 3x2

maka akan memiliki turunan:
f'(x) = 6x
nah, dengan Integral, kita dapat 'menaikkan' kembali birangan tersebut, yaitu dengan rumus
\int axn dx = a/n+1 . xn+1 + C
maka akan diperoleh:
\int 6x dx = 6/1+1 . x1+1 + C
= 6/2 . x2 + C
= 3x2 + C
Keterangan :
\int = Lambang integral
C = Konstanta, bilangan pasti yang belum ditentukan.


Praktik dalam Soal
  • Pada bentuk perkalian/bembagian
    misal,
    f'(x) = x2 . x1/2

    maka lebih baik diubah dalam bentuk :
    f(x) = x5/2
    Untuk mempermudah penghitungan integral.
  • Pada bentuk penjumlahan/pengurangan
    misal,
    f'(x) = x5/2 + x2

    maka peng-integralan dilakukan dengan sendiri - sendiri, sehingga menjadi:
    f(x) = 1/5x5/2 + 1/3x3 + C
    Keterangan :
    Hasil dari C + C = C
